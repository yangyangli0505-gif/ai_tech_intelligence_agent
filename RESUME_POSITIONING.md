# Resume Positioning

## 项目一句话

基于长期运行 Agent runtime 二次开发的 AI 技术情报跟踪助手，持续监控 AI / LLM / Agent 生态多源信息，识别高价值变化并主动生成日报、复盘与专题追踪。

## 简历亮点拆解

- 多源信息抓取与统一事件建模
- 去重、变化检测、主题分类、信号排序
- 长期运行的 cron + 主动推送 workflow
- 面向 AI / LLM / Agent 领域的垂直情报场景
- 基于现有 Agent runtime 做二次开发，而不是只写 prompt

## 面试讲法关键词

- intelligence pipeline
- multi-source monitoring
- change detection
- topic tracking
- signal ranking
- proactive delivery
- long-running agent runtime

## 和 myagent-new-v3 的互补关系

- myagent-new-v3：从零搭建 agent harness 原型，强调权限、工具、RAG、MCP、compression
- AI Tech Intelligence Agent：基于成熟 runtime 的场景化产品开发，强调长期运行、情报分析、主动交付
